Why FynMesh?

A modern approach to building scalable enterprise applications with micro-frontends

Module Federation

Share code and dependencies between applications at runtime using Rollup-based Module Federation with SystemJS.

Multi-Framework Support

Run React, Vue, Preact, Solid, Svelte, and Marko applications side by side in the same shell.

Independent Deployment

Each micro-frontend can be developed, tested, and deployed independently by different teams.

Middleware System

Extensible middleware architecture for cross-cutting concerns like authentication, theming, and layouts.

Multi-Version Support

Run multiple versions of the same library simultaneously. Use React 18 and React 19 in different FynApps.

Smart Dependency Sharing

Intelligent sharing of dependencies across FynApps with version negotiation and deduplication.

Framework Agnostic

Build micro-frontends with your preferred framework - they all work together seamlessly
